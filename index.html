Notice d'utilisation
Vue principale — Synoptique
Tap tuile Ouvre le panneau de modification du véhicule (statut, équipage, mission).
Compteurs Tap sur un compteur en haut (Disponibles, Engagés…) pour filtrer la vue. Retap pour tout réafficher.
Légende Rappel des codes couleur actifs : barre verte = disponible, jaune = engagé, rouge = indispo mécanique, violet = équipage, orange = garde non assurée, bleu = admin, rose = transfert.
Modifier un véhicule
Statut Disponible Engagé Indisponible
Indisponible Si indisponible, précise le type : Mécanique, Équipage, Garde, Admin, Transfert ou Autre. Ajoute un motif visible sur la tuile et un retour prévu.
Mission Sélectionne le type : Primaire, Secondaire, TIIH ou Hélico. Pour Secondaire, TIIH et Hélico, les champs Départ → Arrivée apparaissent automatiquement.
Équipage Renseigne Médecin, IDE et Ambulancier. Les noms saisis sont mémorisés automatiquement dans la liste pour les prochaines fois. Tu peux aussi saisir un nom libre.
Valider Enregistre les modifications et met à jour la tuile immédiatement.
Gérer les engins et bases
⚙ Gérer Ouvre le panneau de gestion. Permet d'ajouter, renommer ou supprimer des engins et des bases/sites.
⠿ Glisser Maintenir le ⠿ et glisser pour réordonner les véhicules. L'ordre est reflété dans le synoptique après avoir cliqué Enregistrer.
Bases Chaque base correspond à un site géographique (Bourges, Vierzon, Saint-Amand…). Les bases avec beaucoup d'engins s'affichent automatiquement à gauche, les petites à droite.
× Supprimer Un engin peut être supprimé depuis la liste. Une base ne peut être supprimée que si elle ne contient plus d'engins.
Synchronisation
LOCAL Les données sont sauvegardées localement dans le navigateur. Chaque poste a sa propre version.
LIVE Quand Firebase est configuré, toutes les modifications sont synchronisées en temps réel sur tous les postes. Le badge passe au vert.
